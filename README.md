# PNALM Biodiversity Dataset Collection

This repository is the central hub for a collection of standardized biodiversity datasets from the Parco Nazionale d'Abruzzo, Lazio e Molise (PNALM).

Each dataset is maintained in a separate repository and focuses on a specific taxonomic group.

## Available datasets

See datasets.md

## Goal

To provide standardized, reusable biodiversity datasets for ecological and evolutionary research.

## Structure

Each dataset repository follows a shared structure and data standard.

## Standards

See the `standards/` folder.
